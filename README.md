# Тестовое задание для MirrorApp
## Стек:
- Python 3.11
- FastAPI
- TortoiseORM
- Postgres
- SQlite
- Docker

## Запуск

1. Переименовать .env.example -> .env
2. Запустить докер контейнер `docker-compose up --build`
